# Mod Management

VoxVer Launcher integrates CurseForge + Modrinth dual sources, supporting one-stop mod searching, downloading, and management.

## Search & Download

1. Go to the **Downloads** page
2. Search for a mod by name, or browse by category
3. Select the mod you need and click "Download"
4. Choose the instance to install it into
5. Wait for the download to complete

> Download progress can be viewed in the global download popup at the bottom right.

## Local Mod Management

Go to the **Instance Details** page to see the list of installed mods for that instance:

| Action | Description |
|--------|-------------|
| Enable / Disable | Toggle the switch without deleting the file |
| Config Editing | Supports editing mod configuration files in toml / json format |
| Delete | Remove the mod file from the instance |
| Update Detection | Detects updates based on Modrinth SHA1 hash |

## ModLoader Support

| ModLoader | Description |
|-----------|-------------|
| Fabric | Large mod support, lightweight |
| Forge | Well-established, good compatibility |
| NeoForge | The new successor to Forge |
| Quilt | A fork of Fabric, more modular |

## FAQ

**Mods not working?** Confirm that the ModLoader is correctly installed and that the game version matches the mod's supported version.

**Slow download speed?** Go to **Settings** to switch the download source.
