# Quick Start

This guide walks you through launching the game for the first time.

## Step Overview

```
Install VoxVer Launcher
  -> Add Account (Microsoft Login / Offline Mode)
  -> Install Game Version
  -> Configure Java (optional, auto-detection supported)
  -> Click Launch
```

## 1. Add Account

Open VoxVer Launcher and go to the **Accounts** page:

- **Microsoft Login**: Click "Microsoft Login". A browser window will open where you can sign in with your Microsoft account credentials to authorize.
- **Offline Mode**: Enter any username. Suitable for single-player gameplay.

## 2. Install Game Version

Go to the **Versions** page:

1. Click "Refresh Version List"
2. Select the desired version (supports releases / snapshots / old versions / April Fools versions)
3. Click "Install"

The first installation will download core game files. Please be patient.

## 3. Install ModLoader (Optional)

If you want to use mods:

1. Go to the **Versions** page
2. Select an installed vanilla version
3. Click "Install ModLoader" and choose Fabric / Forge / NeoForge / Quilt

## 4. Launch Game

Return to the **Launch** page:

1. Select the version
2. Click "Launch Game"
3. Wait for the progress bar to complete; the game window will open automatically

## Common Issues

**Launch failed?** Check [Crash Monitor](/en/guide/crash) to learn how to view error logs.

**Java error?** Go to **Settings** and manually specify the Java path. See [Java Settings](/en/guide/java).
