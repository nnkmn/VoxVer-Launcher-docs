# Global Settings

MCLa provides a variety of global configuration options to meet your personalization needs.

## General Settings

| Setting | Description |
|---------|-------------|
| Game Directory | Default is `.minecraft`, can be customized |
| Download Threads | Controls the number of concurrent downloads |
| Download Source | Can select BMCLAPI and other mirrors |
| Minimize After Launch | Automatically minimize the launcher after launching the game |
| Close After Launch | Automatically close the launcher after launching the game |

## Appearance Settings

| Setting | Description |
|---------|-------------|
| Theme | Dark theme / Light theme |
| Font Size | Adjust the interface font size |
| Pixel UI | Enables the Press Start 2P pixel font when turned on |

## Advanced Settings

| Setting | Description |
|---------|-------------|
| Custom JVM Arguments | Additional launch parameters |
| GC Optimization | Enables default GC optimization parameters |
| Log Level | DEBUG / INFO / WARN / ERROR |

## Reset Settings

Go to the **Settings** page, scroll to the bottom, and click "Restore Default Settings" to reset all configurations.
