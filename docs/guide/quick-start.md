# 快速启动

本章节将引导你在五步之内完成首次游戏启动。如果你是第一次使用 VoxVer Launcher，请按照以下流程操作。

## 步骤概览

```
安装 VoxVer Launcher
    |
    v
第一步：添加账户（微软正版登录 / 离线模式）
    |
    v
第二步：安装游戏版本（刷新列表 -> 选择版本 -> 安装）
    |
    v
第三步：安装 ModLoader（可选，仅在使用 Mod 时需要）
    |
    v
第四步：配置 Java（可选，通常自动检测即可）
    |
    v
第五步：启动游戏
```

## 第一步：添加账户

打开 VoxVer Launcher，进入左侧导航栏的 **账户** 页面。VoxVer Launcher 支持两种登录方式，请根据你的需求选择：

### 微软正版登录

如果你拥有 Minecraft Java 版正版账号，推荐使用微软登录：

1. 在「账户」页面点击 **「微软登录」** 按钮
2. 启动器会自动打开默认浏览器，并显示一个设备码（例如 `A1B2C3D4`）
3. 在浏览器中登录你的微软账号（如尚未登录）
4. 前往 [microsoft.com/devicelogin](https://microsoft.com/devicelogin)，输入页面显示的设备码
5. 按照页面提示完成授权
6. 回到 VoxVer Launcher，启动器会自动获取并保存访问令牌（token）
7. 登录成功后，你的玩家头像和用户名会显示在账户列表中

::: tip 提示
首次登录后，VoxVer Launcher 会自动刷新 token，后续无需重复登录。详细说明请参阅 [账户管理](/guide/accounts)。
:::

### 离线模式

如果你只需要单机游玩或进行测试，可以使用离线模式：

1. 在「账户」页面点击 **「离线登录」** 按钮
2. 在弹出的对话框中输入你想要的用户名（任意英文字符）
3. 点击「确认」完成登录

::: warning 注意
离线模式无法连接正版服务器（如 Hypixel），仅适用于单机游戏或局域网联机。
:::

## 第二步：安装游戏版本

进入左侧导航栏的 **版本** 页面：

1. 点击 **「刷新版本列表」** 按钮，VoxVer Launcher 会从 BMCLAPI 镜像源获取最新的版本列表
2. 在列表中选择你需要的游戏版本：
   - **正式版（Releases）** -- 稳定版本，推荐大多数玩家使用
   - **快照（Snapshots）** -- 开发中的测试版本，可能存在 Bug
   - **远古版（Old Versions）** -- 早期历史版本
   - **愚人节版（April Fools）** -- 每年愚人节特别版本
3. 选定版本后，点击 **「安装」** 按钮
4. 等待下载完成，安装进度会在右下角的全局下载弹窗中显示

::: info 首次下载说明
首次安装游戏版本时，VoxVer Launcher 需要下载游戏核心文件、资源文件和库文件，总大小通常在 200MB - 500MB 之间，具体取决于版本。请确保网络连接稳定，耐心等待下载完成。
:::

## 第三步：安装 ModLoader（可选）

如果你计划使用 Mod，需要先安装对应的 ModLoader：

1. 在「版本」页面找到已安装的原版游戏
2. 点击该版本旁的 **「安装 ModLoader」** 按钮
3. 选择你需要的 ModLoader：

| ModLoader | 特点 | 推荐场景 |
|-----------|------|---------|
| Fabric | 轻量级，加载速度快，兼容大量 Mod | 追求性能和兼容性的玩家 |
| Forge | 老牌加载器，生态成熟，大型 Mod 支持好 | 需要使用大型整合包的玩家 |
| NeoForge | Forge 的新继任者，面向新版本 | 1.20+ 版本推荐 |
| Quilt | Fabric 的分叉，更模块化 | 需要高级模块化功能的玩家 |

4. 选择版本后，点击确认，等待安装完成

## 第四步：配置 Java（可选）

VoxVer Launcher 会在首次启动时自动检测系统中已安装的 Java 环境。如果检测成功，你可以跳过此步骤。

如果自动检测失败，或你需要使用特定版本的 Java，请参阅 [Java 设置](/guide/java) 进行手动配置。

## 第五步：启动游戏

回到 VoxVer Launcher 的 **首页（启动）** 页面：

1. 确认已选择正确的游戏账户（在账户下拉列表中）
2. 确认已选择正确的游戏版本
3. 点击 **「启动游戏」** 按钮
4. 等待启动进度条完成，游戏窗口会自动弹出

::: tip 提示
启动过程中，VoxVer Launcher 会自动验证游戏文件完整性、下载缺失的依赖库，并拼装正确的启动参数。整个过程通常需要 5-30 秒。
:::

## 遇到问题？

- **启动失败或闪退？** 请查看 [崩溃监控](/guide/crash) 了解如何查看错误日志和排查原因。
- **Java 报错？** 请进入「设置」页面手动指定 Java 路径，详见 [Java 设置](/guide/java)。
- **其他问题？** 请查阅 [常见问题](/guide/faq)。
