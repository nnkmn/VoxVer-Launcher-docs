# 全局设置

VoxVer Launcher 提供丰富的全局配置选项。自 v0.6.0 起，设置页面按功能分类拆分为多个子页面，顶部分为「游戏」「个性化」「网络」「其他」四大类别，每类下包含若干子页面，方便快速定位所需配置。

## 设置页结构

| 顶部分类 | 子页面 | 说明 |
|---------|--------|------|
| **游戏** | 主页、账户、关于、版权、游戏账户与档案、Java虚拟机与内存、游戏目录、高级设置 | 游戏运行、账户管理与启动核心配置 |
| **个性化** | 主题与背景、主界面、语言、辅助功能 | 界面外观、主题、语言与无障碍设置 |
| **网络** | 下载、联机、安全识别服务 | 下载源、联机服务与认证配置 |
| **其他** | 服务与反馈、赞助我们、开发者选项 | 反馈渠道、赞助与开发者工具 |

::: tip 导航方式
进入 **设置** 页面后，左侧边栏列出当前分类下的所有子页面，点击即可切换。顶部标签栏用于在四大类别之间切换。
:::

## 游戏目录

**路径**：设置 → 游戏 → 游戏目录

指定 Minecraft 游戏文件的存储位置。所有版本、Mod、存档等数据都存储在此目录下。修改后需要重启 VoxVer Launcher 才能生效。详见 [自定义游戏路径](/guide/custom-path)。

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 游戏目录 | `~/.minecraft` | 任意有效路径 |

## 下载设置

**路径**：设置 → 网络 → 下载

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 下载线程数 | 并发下载的最大线程数量 | 32 |
| 下载源 | 游戏文件和资源的下载镜像源 | BMCLAPI |

### 下载线程数

控制同时下载文件的最大并发数。数值越大，下载速度理论上越快，但过高可能导致网络拥堵或下载失败。建议根据网络带宽调整：
- 宽带 100Mbps 以下：16-32 线程
- 宽带 100-500Mbps：32-64 线程
- 宽带 500Mbps 以上：64-128 线程

### 下载源

选择游戏文件和资源的下载服务器。国内用户推荐使用 BMCLAPI 镜像源，下载速度更快。

::: info v0.6.1 下载源优化
自 v0.6.1 起，资源文件（assets）的主下载源改为官方 Minecraft 资源服务器（`resources.download.minecraft.net`，无速率限制），BMCLAPI 作为备用源。此优化修复了此前 3300+ 资源文件因 BMCLAPI 速率限制（60秒10次）全部返回 403 的问题。游戏核心文件仍使用你在设置中选择的下载源。
:::

## Java 虚拟机与内存

**路径**：设置 → 游戏 → Java虚拟机与内存

此处配置 Java 运行环境与内存分配，详见 [Java 设置](/guide/java)。

## 高级设置

**路径**：设置 → 游戏 → 高级设置

高级设置面向有经验的用户，提供 JVM 参数和日志相关的配置。

| 设置项 | 说明 | 默认值 | 注意事项 |
|--------|------|--------|---------|
| 自定义 JVM 参数 | 附加到游戏启动命令中的额外 Java 虚拟机参数 | 空 | 需要了解 JVM 参数语法 |
| 日志级别 | 启动器日志的输出详细程度 | INFO | DEBUG 级别会产生大量日志 |

### 自定义 JVM 参数

允许你向游戏启动命令中追加额外的 JVM 参数。例如：
- `-XX:+UnlockExperimentalVMOptions` -- 解锁实验性 VM 选项
- `-XX:MaxGCPauseMillis=50` -- 设置 GC 最大停顿时间

::: warning 注意
- 仅在了解 JVM 参数含义的情况下修改此项。
- 错误的 JVM 参数可能导致游戏无法启动或运行不稳定。
- 如果修改后出现问题，请清空此字段并重启 VoxVer Launcher。
:::

### GC 垃圾回收（自动）

::: info v0.6.1 变更
自 v0.6.1 起，VoxVer Launcher **自动**为所有游戏版本启用 G1GC 垃圾回收器，无需手动配置：
- **1.18+ 版本**：`-XX:+UseG1GC` + `MaxGCPauseMillis=50` + `DisableExplicitGC`
- **1.17 及以下版本**：`-XX:+UseG1GC` + `MaxGCPauseMillis=100` + `DisableExplicitGC`

此前版本中可选的 CMS GC 已被移除（Java 14+ 已不再支持），统一使用 G1GC 以保证兼容性。
:::

### 日志级别

控制 VoxVer Launcher 启动器自身日志的详细程度：

| 级别 | 说明 | 适用场景 |
|------|------|---------|
| DEBUG | 输出所有调试信息 | 开发调试、排查问题 |
| INFO | 输出一般信息（默认） | 日常使用 |
| WARN | 仅输出警告和错误 | 仅关注异常情况 |
| ERROR | 仅输出错误 | 最小化日志输出 |

## 主题与背景

**路径**：设置 → 个性化 → 主题与背景

外观设置允许你自定义 VoxVer Launcher 的界面风格。

| 设置项 | 说明 | 默认值 | 可选值 |
|--------|------|--------|--------|
| 主题 | 界面配色方案 | 深色主题 | 深色主题 / 浅色主题 |
| 背景图片 | 自定义启动器背景图 | 默认背景 | 任意图片 |
| 背景模糊 | 背景图模糊程度 | 0px | 0 - 20px |
| 背景暗化 | 背景图暗化程度 | 0% | 0% - 100% |

### 主题切换

切换深色和浅色主题。深色主题适合暗光环境使用，浅色主题适合明亮环境。自 v0.6.0 起，主题系统完成重构，支持完整的浅色/深色覆盖样式。

### 背景自定义

自 v0.5.5 起支持背景自定义，可选择本地图片作为启动器背景，并调整模糊和暗化程度以达到最佳可读性。

## 主界面

**路径**：设置 → 个性化 → 主界面

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 字体大小 | 界面文字的显示大小 | 默认 |
| 像素风 UI | 使用像素风格字体和界面元素 | 关闭 |
| 动画效果 | 界面过渡动画 | 开启 |
| 特效开关 | 粒子等视觉特效 | 开启 |
| 音效开关 | 界面操作音效 | 开启 |

### 字体大小

调整界面中所有文字的大小。如果你使用高分辨率显示器，可以选择「大」或「特大」以获得更好的可读性。

### 像素风 UI

开启后，界面字体会切换为 **Press Start 2P** 像素字体，整体风格更加贴近 Minecraft 的像素艺术风格。适合喜欢复古像素风的玩家。

## 语言

**路径**：设置 → 个性化 → 语言

切换界面显示语言。支持简体中文和 English，切换后界面实时更新。详见 [国际化](#) 说明。

## 辅助功能

**路径**：设置 → 个性化 → 辅助功能

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 屏幕阅读器模式 | 增强屏幕阅读器兼容性，优化焦点导航 | 关闭 |
| 减少动画效果 | 禁用界面过渡动画和粒子特效 | 关闭 |
| 高对比度模式 | 提升界面元素对比度，便于视障用户 | 关闭 |

## 重置设置

如果你希望恢复所有设置到默认状态：

1. 进入 **设置** 页面
2. 滚动到页面底部
3. 点击 **「恢复默认设置」** 按钮
4. 在确认对话框中点击「确认」

::: warning 注意
重置设置会将所有配置项恢复为默认值，包括游戏目录、Java 路径、外观设置等。重置后需要重新配置。账户信息和已安装的游戏版本不受影响。
:::

## 主题自定义

VoxVer Launcher 提供 **色盘编辑器**，允许你自定义启动器的主题配色方案，打造个性化的界面风格。

### 打开主题编辑器

1. 进入 **设置** 页面
2. 在「外观设置」区域点击 **「主题编辑器」** 按钮
3. 弹出色盘编辑器，可以调整各项颜色

### 可定制的颜色

| 颜色项 | 作用 | 调整效果 |
|--------|------|---------|
| 主色 | 启动器品牌色 | 顶部导航栏、按钮、链接 |
| 辅助色 | 强调色 | 选中状态、激活按钮 |
| 背景色 | 主背景 | 应用整体背景 |
| 文字色 | 主要文字 | 标题、正文 |
| 边框色 | 分割线、卡片边框 | UI 分割元素 |

### CSS 变量动态计算

主题色选定后，VoxVer Launcher 会基于色彩理论自动计算相关的次级颜色：

```
选定主色 → 计算深浅变化 → 自动生成 5 级色阶
                                       |
                                       v
                    自动应用到：--vp-c-brand-1 ~ --vp-c-brand-5
```

::: tip 提示
- 主题设置会立即生效，实时预览。
- 可保存多套主题方案，命名后在「主题」下拉框中快速切换。
- 主题文件存放在本地，可导出分享给其他用户。
:::

## 全局快捷键

VoxVer Launcher 支持自定义 **全局快捷键**，无需聚焦到启动器即可触发常用操作。

### 默认快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + Alt + L` | 显示 / 隐藏启动器主窗口 | 全局生效 |
| `Ctrl + Alt + G` | 启动游戏 | 直接启动当前选中的游戏 |
| `Ctrl + Alt + D` | 打开下载管理 | 全局唤出下载弹窗 |

### 自定义快捷键

1. 进入 **设置** 页面
2. 找到「快捷键」区域
3. 点击要修改的快捷键对应的输入框
4. 按下新的组合键（支持 `Ctrl`、`Alt`、`Shift`、`Meta` + 字母/功能键）
5. 点击「保存」即可生效

::: warning 注意
- 避免与系统或其他常用软件的快捷键冲突。
- 部分游戏在全屏模式下可能无法接收全局快捷键。
:::
