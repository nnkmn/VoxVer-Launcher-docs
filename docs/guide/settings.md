# 全局设置

VoxVer Launcher 提供丰富的全局配置选项，涵盖常规、外观和高级设置三大类别，满足不同用户的个性化需求。本章将逐一介绍各项设置的功能和使用方法。

## 常规设置

常规设置包含游戏运行和下载相关的核心配置。

| 设置项 | 说明 | 默认值 | 可选值 |
|--------|------|--------|--------|
| 游戏目录 | Minecraft 游戏文件的存储根目录 | `~/.minecraft` | 任意有效路径 |
| 下载线程数 | 并发下载的最大线程数量 | 32 | 1 - 128 |
| 下载源 | 游戏文件和资源的下载镜像源 | BMCLAPI | BMCLAPI / 官方源 |
| 启动后最小化 | 游戏启动后自动最小化启动器窗口 | 关闭 | 开启 / 关闭 |
| 关闭启动器 | 游戏启动后自动关闭启动器 | 关闭 | 开启 / 关闭 |

### 详细说明

**游戏目录** -- 指定 Minecraft 游戏文件的存储位置。所有版本、Mod、存档等数据都存储在此目录下。修改后需要重启 VoxVer Launcher 才能生效。详见 [自定义游戏路径](/guide/custom-path)。

**下载线程数** -- 控制同时下载文件的最大并发数。数值越大，下载速度理论上越快，但过高可能导致网络拥堵或下载失败。建议根据网络带宽调整：
- 宽带 100Mbps 以下：16-32 线程
- 宽带 100-500Mbps：32-64 线程
- 宽带 500Mbps 以上：64-128 线程

**下载源** -- 选择游戏文件和资源的下载服务器。国内用户推荐使用 BMCLAPI 镜像源，下载速度更快。

**启动后最小化** -- 开启后，点击「启动游戏」时，VoxVer Launcher 窗口会自动最小化到任务栏，而不是保持在前台。

**关闭启动器** -- 开启后，点击「启动游戏」时，VoxVer Launcher 会在游戏启动后自动关闭，释放系统资源。如需重新打开，请从开始菜单或桌面快捷方式启动。

## 外观设置

外观设置允许你自定义 VoxVer Launcher 的界面风格。

| 设置项 | 说明 | 默认值 | 可选值 |
|--------|------|--------|--------|
| 主题 | 界面配色方案 | 深色主题 | 深色主题 / 浅色主题 |
| 字体大小 | 界面文字的显示大小 | 默认 | 小 / 默认 / 大 / 特大 |
| 像素风 UI | 使用像素风格字体和界面元素 | 关闭 | 开启 / 关闭 |

### 详细说明

**主题** -- 切换深色和浅色主题。深色主题适合暗光环境使用，浅色主题适合明亮环境。

**字体大小** -- 调整界面中所有文字的大小。如果你使用高分辨率显示器，可以选择「大」或「特大」以获得更好的可读性。

**像素风 UI** -- 开启后，界面字体会切换为 **Press Start 2P** 像素字体，整体风格更加贴近 Minecraft 的像素艺术风格。适合喜欢复古像素风的玩家。

## 高级设置

高级设置面向有经验的用户，提供 JVM 参数和日志相关的配置。

| 设置项 | 说明 | 默认值 | 注意事项 |
|--------|------|--------|---------|
| 自定义 JVM 参数 | 附加到游戏启动命令中的额外 Java 虚拟机参数 | 空 | 需要了解 JVM 参数语法 |
| GC 优化 | 启用 VoxVer Launcher 预设的垃圾回收优化参数 | 关闭 | 部分整合包可能与 GC 参数不兼容 |
| 日志级别 | 启动器日志的输出详细程度 | INFO | DEBUG 级别会产生大量日志 |

### 详细说明

**自定义 JVM 参数** -- 允许你向游戏启动命令中追加额外的 JVM 参数。例如：
- `-XX:+UseG1GC` -- 使用 G1 垃圾回收器
- `-XX:+UnlockExperimentalVMOptions` -- 解锁实验性 VM 选项
- `-XX:MaxGCPauseMillis=50` -- 设置 GC 最大停顿时间

::: warning 注意
- 仅在了解 JVM 参数含义的情况下修改此项。
- 错误的 JVM 参数可能导致游戏无法启动或运行不稳定。
- 如果修改后出现问题，请清空此字段并重启 VoxVer Launcher。
:::

**GC 优化** -- 开启后，VoxVer Launcher 会自动为游戏添加经过优化的垃圾回收参数，通常可以减少游戏运行中的卡顿（GC 停顿）。预设参数针对常见场景进行了调优。

**日志级别** -- 控制 VoxVer Launcher 启动器自身日志的详细程度：

| 级别 | 说明 | 适用场景 |
|------|------|---------|
| DEBUG | 输出所有调试信息 | 开发调试、排查问题 |
| INFO | 输出一般信息（默认） | 日常使用 |
| WARN | 仅输出警告和错误 | 仅关注异常情况 |
| ERROR | 仅输出错误 | 最小化日志输出 |

## 重置设置

如果你希望恢复所有设置到默认状态：

1. 进入 **设置** 页面
2. 滚动到页面底部
3. 点击 **「恢复默认设置」** 按钮
4. 在确认对话框中点击「确认」

::: warning 注意
重置设置会将所有配置项恢复为默认值，包括游戏目录、Java 路径、外观设置等。重置后需要重新配置。账户信息和已安装的游戏版本不受影响。
:::

## 主题自定义

VoxVer Launcher 提供 **色盘编辑器**，允许你自定义启动器的主题配色方案，打造个性化的界面风格。

### 打开主题编辑器

1. 进入 **设置** 页面
2. 在「外观设置」区域点击 **「主题编辑器」** 按钮
3. 弹出色盘编辑器，可以调整各项颜色

### 可定制的颜色

| 颜色项 | 作用 | 调整效果 |
|--------|------|---------|
| 主色 | 启动器品牌色 | 顶部导航栏、按钮、链接 |
| 辅助色 | 强调色 | 选中状态、激活按钮 |
| 背景色 | 主背景 | 应用整体背景 |
| 文字色 | 主要文字 | 标题、正文 |
| 边框色 | 分割线、卡片边框 | UI 分割元素 |

### CSS 变量动态计算

主题色选定后，VoxVer Launcher 会基于色彩理论自动计算相关的次级颜色：

```
选定主色 → 计算深浅变化 → 自动生成 5 级色阶
                                       |
                                       v
                    自动应用到：--vp-c-brand-1 ~ --vp-c-brand-5
```

::: tip 提示
- 主题设置会立即生效，实时预览。
- 可保存多套主题方案，命名后在「主题」下拉框中快速切换。
- 主题文件存放在本地，可导出分享给其他用户。
:::

## 全局快捷键

VoxVer Launcher 支持自定义 **全局快捷键**，无需聚焦到启动器即可触发常用操作。

### 默认快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + Alt + L` | 显示 / 隐藏启动器主窗口 | 全局生效 |
| `Ctrl + Alt + G` | 启动游戏 | 直接启动当前选中的游戏 |
| `Ctrl + Alt + D` | 打开下载管理 | 全局唤出下载弹窗 |

### 自定义快捷键

1. 进入 **设置** 页面
2. 找到「快捷键」区域
3. 点击要修改的快捷键对应的输入框
4. 按下新的组合键（支持 `Ctrl`、`Alt`、`Shift`、`Meta` + 字母/功能键）
5. 点击「保存」即可生效

::: warning 注意
- 避免与系统或其他常用软件的快捷键冲突。
- 部分游戏在全屏模式下可能无法接收全局快捷键。
:::
